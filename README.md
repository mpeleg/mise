# Mise

A mobile-first recipe management app that removes the friction of capturing, organizing, and sharing recipes.

Speak your recipe after cooking, paste a link, or snap a photo — Mise uses AI to structure it into a clean, searchable, shareable format.

## Project Structure

- `docs/` — Product spec and roadmap
- `mockups/` — UI mockups and design artifacts
- `.claude/` — Claude Code custom commands

## Docs

- [Product Spec](docs/SPEC.md)
- [Build Roadmap](docs/ROADMAP.md)

## Tech Stack

- **Frontend:** React Native (Expo)
- **Backend:** Supabase (Auth, PostgreSQL, Storage, Edge Functions)
- **AI:** OpenAI Whisper (transcription) + Claude (recipe structuring, tagging, captions)
